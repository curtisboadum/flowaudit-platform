# Scaffold API Endpoint

Create a new Next.js API route handler.

## Arguments
$ARGUMENTS

## Steps

1. Determine the resource name and HTTP methods from the arguments or ask the user.
2. Create the route handler at `src/app/api/{resource}/route.ts`.
3. Implement the handler following the `api-endpoint` skill patterns.
4. Add TypeScript types for request/response.
5. Update `docs/api/endpoints.md`.
6. Run `pnpm typecheck` to verify.
7. Update `docs/tracking/change_log.md`.
