# Session Tracking

Update any tracking file in `docs/tracking/`.

## Steps

1. Read the current state of all tracking files:
   - `docs/tracking/session.md`
   - `docs/tracking/change_log.md`
   - `docs/tracking/to-do_list.md`
   - `docs/tracking/checklist.md`
   - `docs/tracking/insights.md`
   - `docs/tracking/decisions.md`
2. Ask which file(s) to update, or infer from context.
3. Make the updates following the format conventions in each file.
4. Confirm the changes made.
