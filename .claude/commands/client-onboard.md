# Client Onboarding

Guided workflow for onboarding a new client.

## Steps

1. Gather client information:
   - Company name
   - Contact email
   - Plan (starter / growth / enterprise)
   - Primary workflow to automate
2. Create the client record in InstantDB.
3. Scaffold a moat bot agent using the `flowaudit-agent` skill.
4. Configure the agent for the client's use case.
5. Test the agent with sample data.
6. Update tracking files with the new client.
