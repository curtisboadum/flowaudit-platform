# Codex Instructions — FlowAudit

## Project
FlowAudit — AI agency platform. Next.js 15 + InstantDB + Claude API.

## Rules
1. Follow the spec EXACTLY.
2. Check CLAUDE.md for architecture details.
3. Handle ALL states: loading, empty, error, success.
4. pnpm for package management (not npm).
5. TypeScript strict mode — zero errors.
6. After implementation: `pnpm tsc --noEmit && pnpm lint && pnpm build`.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript (strict)
- InstantDB
- Tailwind CSS v4 + shadcn/ui
- Anthropic SDK

## File Conventions
- Components: PascalCase in components/
- Routes: app/ directory (file-based)
- Types: types/ directory
- Utilities: lib/

## Don'ts
- No npm (use pnpm)
- No unnecessary packages
- FlowAudit has its OWN design identity — never copy from Traqd
